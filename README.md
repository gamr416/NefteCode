# Нефтекод 2026

Хакатон по предсказанию результатов окислительного теста Даймлера (DOT) для многокомпонентных рецептур смазочных материалов.

## Структура проекта

```
├── data/                  # Датасеты
│   ├── daimler_mixtures_train.csv
│   ├── daimler_mixtures_test.csv
│   └── daimler_component_properties.csv
├── models/                # Обученные модели
├── notebooks/             # Jupyter ноутбуки
│   └── 01_eda.ipynb       # Первичный анализ данных
├── submissions/           # Файлы для submission
├── src/                   # Исходный код
│   └── eda_analysis.py    # Скрипт EDA
├── inference.ipynb        # Ноутбук для инференса
├── eda_report.md          # Отчет по EDA
├── task.txt               # Условия задачи
└── README.md
```

## Описание задачи

Необходимо разработать модель машинного обучения, которая по данным о составе рецептуры, условиях проведения теста и свойствах компонентов предсказывает:

1. **Delta Kin. Viscosity** - относительное изменение вязкости (%)
2. **Oxidation** - степень окисления (A/cm)

## Ключевые требования

- Работа с переменным числом компонентов в рецептуре
- Использование свойств компонентов
- Учет межкомпонентных взаимодействий (синергия/антагонизм)
- Запрещены: градиентный бустинг, случайный лес, деревья решений
- Интерпретируемость (SHAP/LIME)

## Метрика

Средняя ошибка по нормированным значениям с усреднением по двум целевым переменным.
